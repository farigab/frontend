# Product Overview

BragDoc is a professional achievement tracking application that helps users document and showcase their work contributions through GitHub integration and AI-powered analysis.

## Core Features

- **GitHub OAuth Authentication** - Secure login with GitHub accounts
- **GitHub Integration** - Automatic import of commits, pull requests, and issues
- **AI-Powered Analysis** - Generate intelligent summaries with customizable prompts
- **Flexible Time Periods** - Analyze achievements by day, week, month, or custom ranges
- **Visual Reports** - Charts and insights for contributions
- **Progressive Web App** - Installable with offline support

## User Flow

1. User authenticates via GitHub OAuth
2. Connects GitHub Personal Access Token for data import
3. Selects repositories and time period
4. Customizes AI analysis prompt
5. Views generated reports and insights

## Target Users

Developers who need to:
- Track professional accomplishments
- Prepare for performance reviews
- Document contributions for resumes/portfolios
- Analyze productivity patterns
